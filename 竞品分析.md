## 三、五大网站共同特质分析

### 3.1 视觉设计规律

| 规律 | 具体表现 | 代表案例 |
|------|---------|---------|
| **视觉独特性是核心竞争力** | 每个网站都有唯一识别元素，不使用模板 | Bruno Simon 的3D驾车、Renaud 的日本猫咪、Lynn 的年度重设计 |
| **设计风格与职业定位强绑定** | 视觉语言直接表达专业能力 | 开发者偏暗色+代码感、设计师偏视觉冲击、CSS艺术家把网站当作品 |
| **深色/极简主义主导** | 4/5个网站以深色或极简为基调 | brittanychiang（深蓝）、Bruno（深色3D）、Renaud（黑色系） |
| **高质量配图** | 截图、实机展示，有专门的 OG 图 | 所有5个网站都有设计精良的社交分享图 |
| **微交互细节** | hover效果、加载动画、滚动触发 | Brittany 的精致hover、Renaud 的猫咪loading、Adham 的分屏交互 |

### 3.2 内容策略规律

| 规律 | 具体表现 |
|------|---------|
| **"展示"重于"说明"** | 作品集本身即是能力的最佳证明，网站本身就是第一个作品 |
| **技术深度叙事** | 不只展示结果，还解释过程和决策（Renaud 的技术架构文章） |
| **个性化内容提升亲和力** | Adham 的"随机事实"、Lynn 的宠物介绍、Renaud 的猫咪主题 |
| **社会证明（Social Proof）** | 媒体报道、奖项、知名客户、GitHub stars 等可信度要素显著 |
| **内容持续迭代** | 都在持续更新（Adham：2025-05更新，Brittany：2026年新文章，Lynn：每年重设计） |
| **多形态内容输出** | 除作品集外，还有：博客文章、视频课程、开源项目、书籍、演讲 |

### 3.3 导航与信息架构

| 规律 | 具体表现 |
|------|---------|
| **单页滚动为主流** | Brittany、Renaud 都是单页垂直滚动，减少跳出 |
| **多页面结构适合内容丰富型** | Adham、Lynn 因内容量大采用多页，但导航极简 |
| **内容模块化** | About / Work / Experience / Contact 四大模块几乎是标配 |
| **PDF 简历补充** | Brittany 提供可下载 PDF 简历（方便招聘方） |
| **外链策略清晰** | 只链接到经过筛选的可信来源（公司官网、GitHub、主流媒体）|
| **无障碍（Accessibility）考虑** | Skip to Content 链接（Brittany 最明显），体现专业性 |

### 3.4 SEO 最佳实践

### 4.1 首要原则：网站本身就是你的第一件作品

**建议**：

- 避免使用 Squarespace/Wix 通用模板，即便使用框架也要深度定制

### 4.2 内容结构：必备的4个模块

参考5个网站的共同结构：

1. **About / 关于我**：
   - 一句话定位（你是谁，做什么，服务谁）
   - 当前公司/状态
   - 1-2段真实的个人叙事（不是简历堆砌）
   - 1-2个个性化细节（让人记住你是真实的人）

   - 展示过程和决策思考，不只展示结果

3. **Experience / 经历**：
   - 时间线形式，含公司名称（可点击验证）
   - 每段经历：职责 + 成就 + 技术栈标签

4. **Contact / 联系**：
   - 邮箱 + 社交媒体链接（GitHub、LinkedIn、Twitter）
   - 可选：可下载 PDF 简历

### 4.3 SEO 配置清单
