## 三、三大网站共同特质

### 1. 视觉设计规律
| 特质 | 说明 |
|------|------|
| 极简主义 | 大量留白，少即是多，绝不堆砌元素 |
| 深色系为主 | 三者均以深色/暗色作为主色调，彰显高级感 |
| 大字体标题 | Hero 区域超大字号，第一眼抓住注意力 |
| 动效克制精准 | 有动效但不滥用，每个动画有明确目的 |
| 专属插画/OG图 | 每个网站都有独立设计的社交分享图 |

### 2. 内容策略规律
| 特质 | 说明 |
|------|------|
| 一句话定位 | Hero 区有极其精炼的品牌定位句（≤10字） |
| 中英双语 | 全部支持中英文，但各有侧重 |
| 数字背书 | 用具体数字（10年经验、500万融资）建立可信度 |
| 客户背书 | 展示服务过的知名品牌增强信任 |
| 简历/CV 下载 | 均提供可下载的简历/作品集 |

### 3. SEO 最佳实践
| 特质 | 说明 |
|------|------|
| OG 完整配置 | Title / Description / Image 三要素齐全 |
| Twitter Card | 均设置 `summary_large_image` 提升社交可见度 |
| 个人名+中文 | Title 中同时包含英文名和中文名，双语索引 |
| 语言标记 | 正确设置 `lang` 属性，利于搜索引擎理解受众 |
| Googlebot 指令 | 明确允许图片索引（max-image-preview:large） |

### 4. 品牌表达规律
| 特质 | 说明 |
|------|------|
| 名字即品牌 | 个人名/工作室名 = 域名 = 社交账号，高度统一 |
| 中文文化自信 | 不回避中文，反而将中文作为差异化亮点 |
| IP化 | 设计师本人就是品牌IP，有识别度的个人形象 |

## 四、对你的个人网站建设的启示

### 优先级行动清单

**✅ 必做（核心）**
1. **确定一句话定位**：参考「是设计师，也是程序员」「承包品味」，提炼你的核心差异点（城乡规划博士 × 空间算法 × AI工具开发）
2. **中英双语 SEO**：Title 和 Description 同时包含中英文关键词，例如：`Teng Longyin - 城乡规划博士 & 空间数据开发者`
3. **配置完整 OG 标签**：设计一张专属社交分享图（1200×630px），在所有分享场景下保持品牌一致性
4. **数字背书**：将「07年经验」「04种核心算法」「06+专利」等数字在 Hero 区醒目展示

**⚡ 加分项（差异化）**
5. **东方美学融入**：你的背景是传统建筑/点云算法，可以用水墨纹理、传统建筑线条作为设计语言，与背景视频（东方神韵）形成统一叙事
6. **设计简历下载**：提供 PDF 版本的中英双语简历下载
7. **Twitter/X Card**：设置 `twitter:card: summary_large_image` 提升社交传播效果

## 五、总结

三个 SEO 靠前的中式/东方设计风格网站均遵循：**极简视觉 + 精炼文案 + 完整 SEO + 个人IP化** 的核心逻辑。最大的共同点是——**用中文讲好自己的故事，但用国际化的设计语言展示出来**。这对你的城乡规划博士个人网站同样适用：用东方美学（你的视频背景已经在做了）包裹国际化的学术与技术内容，是最具辨识度的路径。
